📊 HR Attrition Excel Dashboard

An interactive HR Analytics Dashboard built in Microsoft Excel to analyze employee attrition, workforce demographics, performance, and satisfaction. This project demonstrates how Excel can be used as a Business Intelligence tool to generate meaningful HR insights through Pivot Tables, Pivot Charts, KPIs, and an organized star-schema data model.

🚀 Project Overview

The dashboard helps HR professionals understand workforce trends by analyzing:

Employee Attrition
Department-wise Employee Distribution
Gender Distribution
Employee Performance
Job Satisfaction
Overtime Analysis
Years at Company
Workforce Demographics

The project follows a Fact-Dimension (Star Schema) approach, making it scalable and suitable for Power BI integration.
